🧾 Plick · Spike impresión web Bluetooth

Valida si una térmica BT genérica es usable desde el navegador (Web Bluetooth BLE) y compara Web Serial / WebUSB. Proyecto throwaway.

1 · Web Bluetooth (BLE)

Empieza por Diagnóstico: si tu impresora no aparece en el chooser pero sí está en los ajustes BT del sistema → es Bluetooth Clásico/SPP → Web Bluetooth no sirve para ella.

1b · Auto-print (simula pedidos por socket)

Empareja primero con Diagnóstico o Imprimir (1 gesto). Luego inicia: debe imprimir solo, sin que toques nada, reconectando si la impresora se duerme. Es el caso real "pedido por socket → imprime".
